4) DIPLOME - Диплом MATLAB
Теплофизическая обратная задачка где необходимо по известной временной температурной функции лишь на одной из границ объекта, а также имея информацию о температуре объекта в начальный момент времени и тепловом потоке построить функцию распределения температур u(x,t) идея заключается в том, что классические методы позволяют формировать температурные поля зная рапределения температур на всех границах объекта, поэтому приходилось искать зависимости, применять аппроксимирующие и регулиризующие методы, сложность заключалась в правильной комбинации и выборе регулирующих параметров, чтобы получить устойчивые решения для общего вида функций. Мной был проведен анализ влияния входных параметров на характер ведения функций и реализован алгоритм поиска оптимальных регулирующих параметров и таким образом получил устойчивый метод решения задачи.
В папке лежат иллюстрации работы программы

5) OS - Работа с памятью и процессами на C#, C++, C
OS2.c - в терминах API Windows создается процесс, программа ждёт его закрытия выводит "bye" и завершает работу.
OS3.cs - имитация планирования процессов (методами FCFS и RR).
OS4.cpp - работа с синхронизацией процессов и потоков при помощи семафора.
OS5.cpp - работа с памятью в терминах API Windows, необходимо было разместить данные в разных кучах.
